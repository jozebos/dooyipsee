# Decisions — Dooyipsee

## [2026-04-17] Architecture
- Next.js 15 App Router + TypeScript + Tailwind CSS v4
- Deploy via @opennextjs/cloudflare to Cloudflare Pages
- AI via OpenRouter server-side API routes (hide key)
- Static JSON for 78-card database (no external DB)
- Blog as static MDX/TS files (no CMS)
- Card images: AI generated, stored in /public initially (move to R2 if bundle too large)
- Theme: Dark + Purple/Violet cosmic
- Language: Thai only
- 3 AI characters as "personas" mapped to 3 models
- 3 spread types: daily (1), three-card (3), celtic-cross (10)
- dooyipsee.com = canonical, duyipsee.com = 301 redirect
